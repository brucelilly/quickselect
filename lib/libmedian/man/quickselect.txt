DDEESSCCRRIIPPTTIIOONN
       The  qquuiicckksseelleecctt  function  implements  multiple  selection  of   order
       statistics.   Given  an  array  ppkk  of  nnkk _s_i_z_e___t elements representing
       0-based order statistic indices, qquuiicckksseelleecctt partially orders the array
       bbaassee  such  that  the  specified  order  statistics are in-place in the
       array.  If ppkk is _N_U_L_L or nnkk is  0UL,  a  full  sort  of  the  array  is
       performed.

       size_t karray[2];

       places the median (nnmmeemmbb odd) or medians  (nnmmeemmbb even)  in  the  middle

AAPPPPLLIICCAATTIIOONN UUSSAAGGEE
       If  the  macro  QQSSOORRTT__FFUUNNCCTTIIOONN__NNAAMMEE  is defined before quickselect.h is
       included when compiling the quickselect.c source,  a  sorting  function
       with  the  same  semantics  as  qqssoorrtt is generated, using the specified
       name.  A library implementation of qqssoorrtt may be generated  by  defining
       QQSSOORRTT__FFUUNNCCTTIIOONN__NNAAMMEE as qqssoorrtt.

RRAATTIIOONNAALLEE
       While  many  libraries  include  a  standard  qqssoorrtt function, the qqssoorrtt
       implementation may tend to quadratic  performance  on  adverse  inputs.
       Few libraries provide a function for selection of order statistics.

BBUUGGSS aanndd CCAAVVEEAATTSS
       quickselect has expected and worst-case linear complexity for finding a
       single  order  statistic.   quickselect  has  expected  and  worst-case
       lineararithmic complexity for sorting.
